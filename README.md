# Personal Portfolio

My personal portfolio site, built from scratch with HTML, CSS, and vanilla JavaScript.

## 🔗 Live Site

[tito-osakwe.github.io/portfolio](https://tito-osakwe.github.io/portfolio/)

## ✨ Features

- Click the name in the header to cycle through five color themes (CSS custom properties + data attributes)
- A soft gradient blob follows the cursor (`requestAnimationFrame` + easing)
- Sections fade in as they enter the viewport (Intersection Observer API)
- Live Lagos time in the footer (`setInterval` + `Intl.DateTimeFormat`)
- Fully responsive

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, transitions, animations)
- Vanilla JavaScript

No frameworks, no build tools. Just the basics, done deliberately.
